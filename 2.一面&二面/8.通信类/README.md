##  什么是同源策略及限制?

*       源:协议,域名,端口::  这三个中有一个不一样就构成了  源不一样也就是 跨域了

        同源策略限制从一个源加载的文档或脚本如何与来自另一个源的资源进行交互.
        这是一个用于隔离潜在恶意文件的关键安全机制.

        限制的几个方面:

        Cookie  LocalStorage 和 IndexDB

        DOM 无法获取

        Ajax 请求不能发送

##  前后端如何通信

*       Ajax    仅限同源下的通讯方式

        WebSocket 不限制同源限制

        CORS(新的通讯标准)    支持同源通信 和 跨域通信
