#   HTTP 协议类
###     HTTP协议的主要特点: 简单快速 - 灵活 - 无连接 - 无状态

###     HTTP报文的组成部分:(请求报文 & 响应报文)
*           请求报文: 请求行 - 请求头 - 空行 - 请求体
                请求行:http方法 - 页面地址- http协议 -版本

                请求头:keyValue值(告诉服务端请求的内容)

                空行: 下一个

                请求体: 数据部分

            响应报文: 状态行 - 响应头 - 空行 - 响应体

                状态行:http方法 ,状态码

###     HTTP方法:
*           GET: 获取资源
*           POST: 传输资源
*           PUT: 更新资源
*           DELETE: 删除资源
*           HEAD: 获取报文首部
