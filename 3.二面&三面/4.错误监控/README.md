##  前段错误的分类:

*       1) 即使运行错误:代码错误

        2) 资源加载错误

##  错误的捕获方式:

*       即使运行错误的捕获方式:

            1) try..catch   2)window.onerror (只能捕获即使加载错误,无法捕获资源加载错误)
        
        资源加载错误:

            1)object.onerror(没有冒泡)
            //例如:图片

            2)performance.getEntries()
            //高级浏览器中:获取到所有已加载的资源加载时长,可以间接拿到没有加载资源 错误

            3)Error事件捕获
            //

        延伸:跨域的 Js 运行错误可以捕获吗,错误提示什么,应该怎么处理?
